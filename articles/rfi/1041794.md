[Covid-19 : 二千万法国人开始执行宵禁](http://www.rfi.fr//cn/%E6%B3%95%E5%9B%BD/20201017-%E5%B0%BC%E5%8F%A4%E6%8B%89-covid-%E7%97%85%E6%AF%92-%E4%BA%8C%E5%8D%83%E4%B8%87%E6%B3%95%E5%9B%BD%E4%BA%BA%E5%BC%80%E5%A7%8B%E6%89%A7%E8%A1%8C%E5%AE%B5%E7%A6%81)

<div>17/10/2020 - 17:16</div><img src="https://s.rfi.fr/media/display/f50f9d74-1041-11eb-ab62-005056a98db9/w:310/p:16x9/ap20290828619024_0.jpg"><p><strong>法国为刹住Covid-19疫情的蔓延，从2020年10月17日周六零点起开始在首都巴黎大区和另外8座大都市实施宵禁，为期四个星期。二千万法国民众被要求在晚上21点至凌晨6点之间呆在家中，以期控制住不断升高的疫情。</strong></p><div class="t-content__body u-clearfix"><p>据本台RFI 周六当天的法文报道，二千万法国人四周内，将不能在晚上21点至凌晨06点之间，上馆子，看戏剧，违者面临135欧元的罚款，不过，夜间工作者例外。</p><p><em>之前相关报道 - <a target="_self" href="https://www.rfi.fr/cn/法国/20201016-尼古拉-covid-法国警察肩负执行宵禁重任">Covid-19 : 法国警察肩负执行宵禁重任</a></em></p><p><strong>有人提出批评</strong></p><p>这一极端措施受到许多批评。它在个人领域限制了自由。反对者注意到，大部分的群发病例发生在业界和教育业，而这恰好不受宵禁的限制。实际上，群发病例只占感染人数的一小部分。大部分新的确诊案例，已无法追踪起源。</p><p><em>之前相关报道 - <a target="_self" href="https://www.rfi.fr/cn/法国/20201014-尼古拉-新冠病毒-法国2021巴黎农展会有变化">Covid-19:法国2021巴黎农展会有变化</a></em></p><p>所以，如果说企业和学校是传播感染病毒的重要地点，那是错误的。如果说，感染是在个人领域内所造成的，那也是错误的。因为，没有答案。大部分确证者的感染源头已无迹可寻。</p><p><strong><a target="_self" href="https://www.rfi.fr/tw/尼古拉">其它</a> / <a target="_self" href="https://www.rfi.fr/cn/社会/20201015-尼古拉-二十国集团再延贫穷国家债务六个月">二十国集团再延贫穷国家债务六个月</a></strong></p><div class="o-self-promo o-self-promo--nl o-self-promo--hidden" data-selfpromo-newsletter></div><div class="o-self-promo o-self-promo--app o-self-promo--hidden" data-selfpromo-app></div></div>
