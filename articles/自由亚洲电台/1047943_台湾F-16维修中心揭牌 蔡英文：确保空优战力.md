[台湾F-16维修中心揭牌 蔡英文：确保空优战力](https://www.rfa.org/mandarin/yataibaodao/gangtai/hx1-08282020090240.html)
------

<p> </p><p>最近，美国同意向台湾出售66架F16V战机，这意味着台湾未来将有200架F-16投入空中防卫。台湾总统蔡英文宣布成立F-16维修中心，强调捍卫主权和区域安全，不是靠卑躬屈膝，而是要推动国防产业发展，要让世界看到台湾守护国土的决心。 <br/> <br/> 就在美中军事动作频发之际，台湾F-16维修中心在台中汉翔宣布成立，蔡英文总统出席揭牌典礼致词强调：“捍卫中华民国台湾主权、维持区域和平稳定，不是靠卑躬屈膝，而是要有坚实的国防；有了维修中心，战机维修所需时间会大幅缩短，妥善率也会大幅提升，确保国防前线的空优战力。”</p><p> </p><ul><li><b><a class="external-link" width="622"></iframe></p><p><i>视频【蔡英文揭牌F-16维修中心】【再强调和平靠国防 国防自主 振兴产业】</i></p><p> </p><p><br/> 蔡英文说：“维修中心成立后，将可释出的能量，优先由国内厂商筹建。未来30年的产值，粗估约有795亿元，每年将可创造600多个就业机会，创造整体产业效益高达2,000亿元。  ”<br/> <br/> 蔡英文指出，维修中心要有一定比例的退伍空军保修人员，借重他们的专长与经验，退伍后仍能发挥所用，强化军民合作。 蔡英文提到，招商说明会已有超过100多家厂商表达参与产制、维修的意愿，相信更多厂商参与、加入，有助提升台湾在国际航太供应链的地位。 <br/> <br/> 汉翔公司总经理马万钧受访提及，长久以来靠美国原厂洛克希德·马丁（Lockheed Martin）支持维修工作，政府希望透过公私协力，协助空军解决F-16系统零件交期长、价格高、使用频率高等“三高”问题。台湾未来将拥有高达200架F-16，若以妥善率7成、8成估算，随时约有40架F-16需要维修。<br/> <br/> 国家政策研究基金会高级助理研究员揭仲，接受自由亚洲电台采访指出，原本所有军机由空军自己维修，空军业务后勤精简后，部份委由民间厂商维修。<br/> <br/> 揭仲说：“汉翔公司这次在F16A/B性能提升，他是主合约商，有性能提升线，所以具有基本的能量。美方在F16A/B性能提升的时候，也移转了一些技术给汉翔，所以部份的维修业务由汉翔来进行，本来就是顺理成章。”</p><p> </p><p><div class="image-inline captioned" style="width:1280px;"><div class="image-caption"><span style="width:1280px;">台湾F-16维修中心在台中汉翔公司揭牌。（路透社）</span><span class="copyright"> </span></div></div><br/> 揭仲提到，蔡英文政府一开始喊出成立“亚太地区维修中心” ，其他国家F-16也送台湾维修，不过他认为这有困难，因为，维修分三级，比较核心和高端的技术与零件，美国一般会要求返美维修，台湾并没有不可取代性。<br/> <br/> 揭仲说：“它有些是最机密的部件，比如说像上面的雷达、发动机，这部份可能要运回美国修；（但是）牵涉到通讯、电子作战、敌我辨别，可能还是由台湾空军自己修；比如说结构等机敏程度比较低的，就由汉翔自己处理。其他国家把飞机送来台湾修的机率就会降低了。因为送到台湾来处理除了有一笔运费，还有中共抗议的问题。”<br/> <br/> 国防部智库、国防安全研究院助理研究员舒孝煌接受自由亚洲电台采访也提及，台湾目标是建立F16国际维修中心，亚洲国家使用F16的目前有韩国、台湾、新加坡、印尼，其中韩国获授权在韩国生产，新加坡有自己维修的能力。美国空军在亚太地区也有布署，像在韩国有两个F16的联队，在日本有一个F16联队，他们未来可能会陆续换装飞机。</p><p> </p><p><div class="image-inline 空军以“凤展专案”为代号，耗资约新台币1100亿元由美国洛克希德·马丁（Lockheed Martin）公司与汉翔合作，将现有142架F-16A/B升级为F-16V（Block20）；另外，空军也以代号“凤翔专案”，同步向美国采购66架全新F-16V（Block70）战机，未来将部署在台东志航基地。<br/> <br/> 舒孝煌指出，建立民间维修中心，对提高国防自主性帮助很大，台湾未来会获得66架F-16 C/D Block 70，如果建立自己强大的维修能量，能确保飞机寿期零组件供应无虞，确保他的妥善率达到一定水准，也可以解决飞机维修跟后勤碰到的问题。<br/> <br/> 舒孝煌提到，汉翔公司是台湾航发研究的重要企业，目前也执行F16性能提升任务。所谓性能提升涵盖雷达系统、座舱、航电系统等。汉翔还是经国号IDF的生产厂商。未来台湾会有多达两百多架F16战机，将是F16使用国排名前几名的国家。<br/> <br/> <br/> 自由亚洲电台记者 夏小华  台北报导 责编 许书婷 申铧  网编 瑞哲</p>
