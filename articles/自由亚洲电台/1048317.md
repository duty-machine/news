[习近平为“内循环”灭火  企业主怨政府没救助](https://www.rfa.org/mandarin/yataibaodao/jingmao/rc-07212020110004.html)
------

<p>在央视新闻联播的画面中消失二十天后，中国国家主席习近平周二露面了。他在一个企业家座谈会上指出，中国以“国内大循环”为主体的经济，“绝不是关起门来封闭运行”。受邀和习近平座谈的企业代表，除了有遭美国制裁的海康威视外，还有美国微软公司及日本松下公司。习近平这是在释放什么信号？<br/> <br/> 一片掌声中，习近平现身在北京召开的企业家座谈会上。<br/> <br/> 中国多地出现洪水灾情以来，央视新闻联播每晚头条仍是习近平做出的重要指示与讲话，或与外国元首通话、通信，但最近一次播出有习近平影像的新闻消息，已是6月30日中共中央政治局第二十一次集体学习会议的相关报道。<br/> <br/> 睽违二十天后，习近平再次现身发声，面对台下全戴着口罩的企业代表及与会者，央视选择播出没戴口罩的习近平讲话片段，他说：“借此机会，我向广大国有企业、民营企业、外资企业、港澳台资企业、个体工商户为疫情防控和经济社会发展作出的贡献，表示衷心的感谢。”<br/> <br/> 新华社则刊发习近平在会中的完整讲话，他提到三大重点：保护和激发市场主体活力、弘扬企业家精神和集中力量办好自己的事。<br/> <br/> 谈到保护和激发市场主体活力时，尽管他又强调“六稳”与“六保”，还提出要落实纾困惠企政策，要求打造市场化、法治化、国际化营商环境，构建“亲清政商关系”与高度重视支持个体工商户发展。</p><p> </p><ul><li><b><a class="external-link" <br/> 他说，像他这样的个体户，今年最需要的就是流动资金应急，但他有一笔贷款申请已一年，至今没批下来。他认为，中国银行业贷款纾困救急，往往是有关系的或是国营企业为优先，他早已经习以为常。</p><p><div class="image-inline captioned" style="width:622px;"><div style="width:622px;"><img 而习近平则听取了七个企业代表的意见。去年遭美国祭出出口管制制裁的海康威视由董事长、党委书记陈宗年出席，还有中国中化集团有限公司董事长、党组书记宁高宁，以及美国企业微软亚太研发集团主席洪小文与日本松下电器（中国）有限公司总裁赵炳弟发言，提出“十四五”规划建设的建议。</p><p>美国认定，人脸识别设备大厂、中国“天眼计划”的执行者海康威视参与新疆再教育营建设、迫害人权的行动，去年遭美国商务部制裁，名列出口管制实体清单。</p><p><div 今年人大会议期间，习近平提出中国经济实行“国内大循环”的概念。后来，中国副总理刘鹤又在上海陆家嘴论坛上发表书面讲话说， “我们仍面临经济下行的较大压力，但形势正逐步向好的方向转变，一个以国内循环为主、国际国内互促的双循环发展的新格局正在形成。”</p><p>中国领导人的这一表述引发经济界很大震动，因为人们认为这意味着中国要倒退到闭关锁国的时代。</p><p>而习近平在这次会议上似乎要灭火。他多次强调，“中国开放的大门不会关闭，只会越开越大。以国内大循环为主体，绝不是关起门来封闭运行。”</p><p><div <b>留外资稳信心 习近平只愿听自己人意见？</b></p><p>继美国后，日本也传出会给予自中国撤资的日本企业补贴。有媒体最近报道，已经有80多家日本企业从中国撤出。<br/> <br/> 人在纽约的中国经济观察人士秦鹏则告诉本台，找来美国与日本有代表性的企业，特意讲这些话，当然是为了稳定外资信心。“中共现在非常担心这些外资企业继续外逃，会摧毁中国制造的（全球）市场地位，导致国民经济衰败。”<br/> <br/> 另一方面，习近平听取海康威视这类企业的意见，秦鹏认为，这显示两个问题：一是中共明知经济内外情势严峻，智囊还是挑习近平喜欢听的说；二是内循环将来会以向海康威视这样的“自己人”为主导。<br/> <br/> “对中共来说，这是自己的企业，就像华为一样，拚命要保他，因为这也是保他政权。一方面他可以搞内循环养着他，另一方面，政权和这些企业是连动的，他觉得这些企业才是自己人。”他说。   习近平在会中还指出，希望企业家要“增强爱国情怀”，还称“企业营销无国界，企业家有祖国”，更强调明年将进入“十四五”时期，是在全面建成小康社会基础上，开启全面建设社会主义现代化国家新征程的第一个五年。<br/> <br/> 秦鹏说，对习近平而言，推动全面建成小康社会是他的政绩，明年开局必须得好，但这个庚子年的形势发展，基础却不牢，习近平可以说是“顾头不顾腚”的状态，想全面顾周全，却是什么都顾不全。</p><p><br/> <br/> 自由亚洲电台记者郑崇生华盛顿报道   责编：申铧   网编：洪伟</p>
