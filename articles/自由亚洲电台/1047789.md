[央视李红“求和说”翻船 国民党取消赴海峡论坛](https://www.rfa.org/mandarin/yataibaodao/gangtai/hx-09142020085643.html)
------

<p> </p><p>国民党原订由前立法院长王金平率团赴厦门参加海峡论坛。但是央视主持人李红说王金平是“到大陆来求和”，在台湾引发对国民党的嘲讽。国民党主席江启臣要求对方道歉。虽然李红改口说是“求和平”，但已无法平息各方质疑。国民党14日宣布不以政党形式参加，这是海峡论坛12年前开始举办以来的首次。<br/> <br/> 中国央视第四频道《海峡两岸》主持人李红“这人来大陆求和”，掀起台湾不分蓝绿跨党派批评之后，国民党14日决议，不以政党形式率团参加，打破海峡论坛纪录。党员个别参加则未禁止。</p><p> </p><ul><li><b><a class="external-link" 总统府发言人张惇涵表示，重点不在于组团的名义或身分如何转换，重点在于两岸交流必须对等和有尊严，国民党应审慎思考，做出符合台湾的主流民意与国家利益的决定。<br/> <br/> 中国自2009年开始举办“海峡论坛”。依往例由全国政协主席参加开幕式，国民党过去11年大都派副主席率团，洪秀柱任主席时曾亲自率团。今年江启臣当选党主席，未设副主席，规划由前立法院长王金平16日率团出席。央视节目主持人李红评论称，王金平是民进党当局的信使，要传话ECFA两岸经济合作架构协议不应中断。李红评论的标题称《台海兵凶战危，这人要来大陆求和》。蔡英文否认要王金平传话。<br/> <br/> <b>李红改口“求和”是“寻求和平”</b><br/> <br/> 李红的“求和说”，引发台湾不分蓝绿白黄各政党批评。国民党主席江启臣呛声央视“先道歉再说”，不少党内青壮派主张不该去自取其辱。前主席连战之子连胜文更大骂李红“无脑主播”、“帮倒忙的傻B”。而过去曾赴北京听习近平演讲的立委吴斯怀则打强心针，称对中华民国有信心，“从不怕被统战，通常都是我去统战他们”。<br/> <br/> 大陆方面试图灭火，李红11日在节目罕见改口称王金平有份量，有助两岸交流。12日晚间她并透过中评社表示自己在节目中的说法“纯属个人，不代表官方”，并解释她“求和”本意指的是“寻‘求’‘和’平。”大陆国台办11日晚间也发出“新闻稿”，以发言人马晓光名义回应说，两岸各界应多做有利于“增进了解、增信释疑”的事，并强调陆方欢迎王金平先生率团与会。</p><p><br/> <div </b></p><p><b>国民党：七年之病不能求三年之艾</b></p><p>左正东强调，“台湾的民心非常地脆弱，因为两岸关系长期的紧张。有一句话说，七年之病不能求三年之艾。（由于）这种长期以来微妙的关系跟脆弱的心理，台湾社会对这个新的说法，还是没有办法完全的释怀。这才是造成我们最后改变决定的原因。”</p><p>国民党中央有人私下透露，民进党的骂声随着李红改口“求和平”转弱，反倒是国民党内主张取消代表团的声浪愈高涨。一来是不认同有“蓝皮绿骨”嫌疑的王金平率团，另外则是反对江启臣，为各自拥护的下任党主席候选人扯现任主席后腿。否则，以李红的改口，和国台办表示的欢迎，都足以显示对岸展现善意修复差错，国民党也打算靠打“两岸牌”突显和民进党的不同，且不让柯文哲的民众党专美于前。不过也有分析指出，国民党捍然拒绝对岸，或许对民调只剩百分之十几的支持度具加分作用。<br/> <br/> 民进党籍立委林俊宪则呛声，被国内舆论炮轰后，国民党最后决定还是不去海峡论坛了，“跪到一半又爬起来，那一开始何必跪呢？”</p><p>而行政院长苏贞昌14日稍早表示，“中国已经定调这个论坛就是统一，而国民党要派人去，却被羞辱成求和；这种情形下，如果国民党还要派团，王前院长还要去，那国人是什么印象，我们就可以想像。”</p><p> </p><p><div
