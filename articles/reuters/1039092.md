[中国央行公布4月LPR连12月持平 继续追随MLF利率锚稳势](https://cn.reuters.com/article/china-april-lpr-mlf-0420-idCNKBS2C7075)
------

<div><i>2021-04-20T02:18:53Z</i></div><p>路透上海4月20日 - 中国央行周二公布的4月LPR（贷款市场报价利率）报价显示，一年期利率为3.85%，五年期以上品种利率4.65%，连续第12个月持稳，继续追随中期借贷便利（MLF）利率平稳走势。</p><p>路透此前对30位金融机构人士的调查显示，九成受访人士认为4月LPR仍不会调整，此前LPR已连续11个月持稳，加之上周央行进行的中期借贷便利（MLF）操作利率亦持稳，料LPR保持不变概率仍大。</p><p>上周，在巨量地方债发行及例行缴税将造成资金巨大需求之际，中国央行4月MLF操作只是接近等量对冲当月到期（含TMLF），继续传递其稳健中性货币姿态，也说明在基本面持续回暖以及随之而来的通胀压力下，央行也不希望市场过于宽松。</p><p>分析人士并认为，地方债发行势必构成巨大的资金缴款需求，结合央行官员近期表态，对于结构性的流动性需求市场也不必过于担心，预计央行届时仍会通过公开市场灵活操作加以应对。</p><p>中国央行2019年8月推出LPR新机制，一年期品种当年内三次下调，累计降幅为16个基点（bp）至4.15%；新增的五年期以上品种2019年8月首次报在4.85%，随后在当年11月下调一次至4.80%。2020年，一年和五年期LPR各下调两次，其中一年期累计下调30个bp，五年期累计下调15bp。</p><p>下图为LPR新机制以来报价历史走势图表：</p><p><a href="https://tmsnrt.rs/3dxk49V">tmsnrt.rs/3dxk49V</a></p><p>以下为LPR新机制报价的各期情况：</p><p>日期 一年 五年以上</p><p>利率 变动 利率 变动</p><p>4月20日 3.85% 0 4.65%</p><p>3月22日 0 4.65%</p><p>2月20日 3.85% 0 4.65% 0</p><p>1月20日 3.85% 0 4.65% 0</p><p>2020年12月21日 3.85% 0 4.65% 0</p><p>11月20日 3.85% 0 4.65% 0</p><p>10月20日 3.85% 0 4.65% 0</p><p>9月21日 3.85% 0 4.65% 0</p><p>8月20日 3.85% 0 4.65% 0</p><p>7月20日 3.85% 0 4.65% 0</p><p>6月22日 3.85% 0 4.65% 0</p><p>5月20日 3.85% 0 4.65% 0</p><p>4月20日 3.85% -20bp 4.65% -10bp</p><p>3月20日 4.05% 0 4.75% 0</p><p>2月20日 4.05% -10bp 4.75% -5bp</p><p>1月20日 4.15% 0 4.80% 0</p><p>2019年12月20日 4.15% 0 4.80% 0</p><p>11月20日 4.15% -5bp 4.80% -5bp</p><p>10月21日 4.20% 0 4.85% 0</p><p>9月20日 4.20% -5bp 4.85% 0</p><p>8月20日 4.25% -6bp 4.85% --</p><p>自2019年8月20日起，全国银行间同业拆借中心于每月20日（遇节假日顺延）9时30分公布贷款市场报价利率。LPR报价行按公开市场操作利率（主要指一年期为主的MLF利率）加点形成的方式，向全国银行间同业拆借中心报价。（完）</p>
