[中国1月LPR随MLF无悬念九连平 “稳”字当先政策利率暂无调整必要](https://cn.reuters.com/article/china-cen-jan-lpr-0120-idCNKBS29P09W)
------

<div><i>2021-01-20T03:30:54Z</i></div><p>路透上海1月20日 - 中国央行周三公布的新年首次LPR（贷款市场报价利率）无悬念连九月持平。市场人士表示，央行政策强调“稳”字当先，MLF利率锚持稳背景下，LPR短期变化的可能性并不高；且实体经济需求亦表明，当前政策利率水平不存在调整的必要。</p><p>央行稍早公布的1月LPR报价显示，一年期利率为3.85%，五年期以上品种利率4.65%，为连续第九个月持稳，符合市场预期，维持与中期借贷便利（MLF）利率稳健走势同步。</p><p>“LPR没变在预期之中吧，央行政策强调一个‘稳’字，LPR调高的概率不大。”兴业证券固收分析师罗雨浓称。</p><p>路透上日对30位金融机构人士的调查显示，此前LPR已连续八个月持稳，加之本月稍早央行进行的MLF操作利率连九月未变，约有近九成的受访人士认为1月LPR仍不会调整。</p><p>光大银行金融市场部分析师周茂华指出，作为LPR报价利率形成基础之一，MLF本月继续平价续做，市场对LPR仍持稳已有预期；此外，目前利率水平合理适度，实体经济融资供需两旺，显示目前利率水平继续对经济复苏构成有力支持。</p><p>不过他同时提到，值得关注的是，实体融资成本与房地产融资、按揭利率之间形成“利差”，可能吸引部分资金违规流入楼市，干扰楼市调控。</p><p>央行官员上周五明确，要坚持“国内优先”，保持好正常货币政策空间的可持续性。目前国内经济已经回到潜在产出水平，企业信贷需求强劲，货币信贷合理增长，说明当前利率水平是合适的；而不论与其他发展中国家相比还是与中国历史上的准备金率相比，目前的存款准备金率水平都不高。</p><p>这也意味着短期内利率和存准率调整的可能性都很小。这跟路透最新季度调查的结果一致，受访分析机构预计2021年全年一年期LPR将继续维持在3.85%不变，一年期存贷款基准利率也将分别维持在1.5%和4.35%不变。</p><p>中国央行2019年8月推出LPR新机制，一年期品种当年内三次下调，累计降幅为16个基点（bp）至4.15%；新增的五年期以上品种2019年8月首次报在4.85%，随后在当年11月下调一次至4.80%。2020年，一年和五年期LPR各下调两次，其中一年期累计下调30个bp，五年期累计下调15bp。</p><p>下图为LPR新机制以来报价历史走势：</p><p><a href="https://tmsnrt.rs/3p06CPp">tmsnrt.rs/3p06CPp</a></p><p>以下为LPR新机制报价的各期情况：</p><p>日期 一年 五年以上</p><p>利率 变动 利率 变动</p><p>1月20日 3.85% 0 4.65% 0</p><p>2020年12月21日 3.85% 0 4.65% 0</p><p>11月20日 3.85% 0 4.65% 0</p><p>10月20日 3.85% 0 4.65% 0</p><p>9月21日 3.85% 0 4.65% 0</p><p>8月20日 3.85% 0 4.65% 0</p><p>7月20日 3.85% 0 4.65% 0</p><p>6月22日 3.85% 0 4.65% 0</p><p>5月20日 3.85% 0 4.65% 0</p><p>4月20日 3.85% -20bp 4.65% -10bp</p><p>3月20日 4.05% 0 4.75% 0</p><p>2月20日 4.05% -10bp 4.75% -5bp</p><p>1月20日 4.15% 0 4.80% 0</p><p>2019年12月20日 4.15% 0 4.80% 0</p><p>11月20日 4.15% -5bp 4.80% -5bp</p><p>10月21日 4.20% 0 4.85% 0</p><p>9月20日 4.20% -5bp 4.85% 0</p><p>8月20日 4.25% -6bp 4.85% --</p><p>自2019年8月20日起，全国银行间同业拆借中心于每月20日（遇节假日顺延）9时30分公布贷款市场报价利率。LPR报价行按公开市场操作利率（主要指一年期为主的MLF利率）加点形成的方式，向全国银行间同业拆借中心报价。（完）</p>
