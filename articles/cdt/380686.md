[无妄斋｜致孤独殉身的义士](https://chinadigitaltimes.net/chinese/667889.html)
------

<p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481742535.png" alt="file"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481742535.png" alt="file"></p><blockquote><p>「一旦制度无法解决纷争，执法以至法治陷入失效，那么最原始的『自助』(Self-help)，就是惟一能够恢复社会正义的手段，回应民众的共同价值与理念诉求。」 — 2019年曾经写下这句，不过委婉的「自助」，似乎无法使人领略其意味。</p></blockquote><p>2021年7月1日，既为中共建党百周年庆典，亦是香港沦陷中国殖民第24年。 与北京的普天同庆迥异，香港却活像戒严一般：继前一天 <a href="https://news.rthk.hk/rthk/ch/component/k2/1598524-20210630.htm"><strong>邹幸彤</strong></a> 被撤保释再度被捕后， <a href="https://www.bbc.com/zhongwen/trad/chinese-news-57678356">七一游行</a> 没有了，维园因 <strong>《</strong> <a href="https://news.rthk.hk/rthk/ch/component/k2/1598651-20210701.htm"><strong>公安条例</strong></a> <strong>》</strong> 被封了，有组织在铜锣湾设街站，也被大批警察盘问，更有部份市民被捕。</p><p>威权祭出的阵仗，如临大敌；歌舞升平背后，尽是疫情与白色恐怖交织而成的惶恐不安。</p><p>此时，一位无名 <a href="https://www.thestandnews.com/society/ab71%E9%8A%85%E9%91%BC%E7%81%A3%E8%AD%A6%E5%93%A1sogo%E5%A4%96%E7%96%91%E4%B8%AD%E5%88%80-%E6%B6%88%E6%81%AF%E6%8C%87%E6%87%B7%E7%96%91%E6%96%BD%E8%A5%B2%E7%94%B7%E5%86%8D%E7%94%A8%E5%88%80%E8%87%AA%E6%8F%92%E5%BF%83%E5%8F%A3%E5%80%92%E5%9C%B0">义士</a> 出现—以刀刺伤警员后自戳胸膛，然而黑警在他受创后并非及时急救，而是将他双手反绑重压地上，任鲜血流了一地。 孤独的他，送院后伤重不治。</p><p>无人知道其出身背景，抱持何种信念，随身并未携带只言片语，甚至连名字也没有留下。 究竟他怀着怎样的心情，刺向敌人以及自己胸口，亦已考究无从。 他恍若一颗流星，划破天际，自身却化为灰烬散落，不留丝毫痕迹。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481540571.png" alt="file"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481540571.png" alt="file"></p><blockquote><p>「死者已矣，再讨论也无补于事，在生的人如何走下去，才是重点。」</p><p>「我并不鼓励如此白白送命，但对烈士致上最高敬意。愿安息！」</p><p>「如果他活着进警署，你认为他的下场会是怎样？所以他过身，其实是好事。何况自插心脏是他的决定，他已经豁出去，尊重他的决定。」</p><p>「那是对暴政的死谏。」</p></blockquote><p>以上评语，是敢怒不敢言的港人真实反应。</p><p>但同时，也有人急于化身 <strong>「捉鬼大师」</strong> ，企图带动事件可疑的风向；或是祭出阴谋论，认定无名义人是党国证成「孤狼式本土恐袭」，旨在加速 <strong>《基本法》</strong> 23条立法的精心安排；或是 <strong>「离地KOLs」</strong> 再度贩卖恐惧，用以说服港人应该高飞远走，哪怕最终香港沦为赤地荒原。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481565311.png" alt="file"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481565311.png" alt="file"></p><p>再多的口水，也无改铁一般的事实：义士拥有无比的勇气与自制，胜过无数光说不练的塘边鹤。 命如蝼蚁，却负重千钧。</p><p>社会充斥不公正，公平、正义等良善价值一律失灵，单凭个人无力与体制颉颃，试问如何自处？ 社会戾气源自中共党国、特区政府的政权暴力，警察不再是维护公共安全的象征，而是肆意殴打、残虐、侵犯市民的执法机关。 此时，「自力救济」是理所当然之事。</p><p>义士并非「报复社会」，更无滥伤无辜，他是明确针对黑警行刺，虽死无悔。 质疑甚至嘲笑勇者的行为，不过是暴露自身的卑劣与丑陋，而已。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481589361.png" alt="file"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481589361.png" alt="file"></p><p>去年杨佳母亲的悼辞</p><p>无须想像义人如 <a href="https://www.thenewslens.com/article/79491"><strong>安重根</strong></a> 般志士忧国，因为公义沦丧、希冀尽失，已是最有力的理由。</p><p>一如2008年的北京青年 <a href="https://www.dw.com/zh/%E6%9D%A8%E4%BD%B3%E6%89%A7%E8%A1%8C%E6%AD%BB%E5%88%91%E4%B8%89%E5%91%A8%E5%B9%B4-%E7%BD%91%E6%B0%91%E8%87%AA%E5%8F%91%E7%BA%AA%E5%BF%B5/a-15559418"><strong>杨佳</strong></a> 。 同样是7月1日，曾遭警察冤枉、侮辱、虐待的杨佳，历经苦诉无门，身心俱疲，终于无奈选择以暴易暴、以杀止杀的做法，杀伤上海市公安分局内警员及保安。 事发后他被判死刑，虽有知识份子、社会人士为其奔走请命，最终仍逃不过死亡的命运，于同年11月处刑。</p><p>他生前曾说过： <strong><em>「有些委屈如果要一辈子背在身上，那我宁愿犯法。任何事情，你要给我一个说法，你不给我一个说法，我就给你一个说法。」</em></strong> 可谓其人生写照，也是上述「自助」的真实面目—并非逞私义以取代公义，而是在正义败坏的当下，凭借良知之裁量，摆脱体制主宰，复归个人意志之觉醒，铸炼成自由之剑与盾，重塑良善正义的社会。</p><p>时至今天，依旧有人默默悼念杨佳，即使许多不支持他的行为，但至少，仍不忘那位为求公道而挺身的他，为那短暂消逝的正义致哀。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481623800.png" alt="file"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481623800.png" alt="file"></p><p>英雄烈士即使化为历史陈迹，往往不能以成败论。 然而之于威权党国，必将设法控制舆论，剥夺悲壮的死亡，污蔑成暴徒、恐怖份子，让他们死得卑微丑恶，背负骂名，尊严尽丧，求仁而不得，反沦为畏罪懦夫，好让愚夫愚妇以处决穷凶极恶的罪犯般的眼光，旁观烈士行刑，呼喊罪有应得。</p><p>但，请记住烈士的姓名： <strong>梁健辉</strong> 。 为这位孤独英雄，与其壮烈的反动，深深默哀。 纵然命消身殒，但强大意志永不毁灭，光复香港的英灵殿上，必有他的一席位。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/07/image-1625481642136.png" alt="file"></noscript><img class="lazyload"
