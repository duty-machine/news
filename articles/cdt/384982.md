[2047通讯社｜专访“Fuck学习强国”刷分工具作者唐文正（Xmader）](https://chinadigitaltimes.net/chinese/663593.html)
------

<p><strong>CDT编者按：</strong>3月12日，<a href="https://2047.name/t/11552" title="2047通讯社">2047通讯社</a>对“Fuck学习强国”项目作者唐文正（Xmader）进行了采访，这一篇采访也被置顶在了“Fuck学习强国”Github页的<a href="https://github.com/fuck-xuexiqiangguo/Fuck-XueXiQiangGuo/issues" title="讨论区">讨论区</a>，访谈中提及<a href="https://china-digital-times.github.io" title="中国数字时代内容抓取备份站点">中国数字时代内容抓取备份站点</a>是唐文正先生创建。中国数字时代编辑部未予证实，但对此帮助传播反审查信息的行为表示感谢！！！！！！同时，唐文正先生在访谈中认为，中国数字时代2021年3月10日关于他的报道中的一段话“大多只是臆测，对我有很多误解”。这段话根据网络信息整理而成，现根据唐文正先生本人提供的信息予以更正。特此说明。</p><p><noscript><img src="https://chinadigitaltimes.net/chinese/files/2021/03/image-1615797616665.png" alt="file"></noscript><img class="lazyload" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20210%20140%22%3E%3C/svg%3E" data-src="https://chinadigitaltimes.net/chinese/files/2021/03/image-1615797616665.png" alt="file"></p><p><strong>现将完整采访转发如下：</strong></p><p>原标题：“Fuck学习强国”作者唐文正(Xmader)专访</p><blockquote><p><a href="https://github.com/fuck-xuexiqiangguo/Fuck-XueXiQiangGuo">Fuck-XueXiQiangGuo</a>（以下简称fuck-xxqg）是一款“学习强国”刷分软件，这个软件被许多中国体制内人士用于完成工作单位强制分配的“学习强国”任务。经过一年多的开发迭代，fuck-xxqg在Github上获得了超过7000颗星星。</p></blockquote><p>《中国数字时代》对fuck-xxqg项目的两篇报道：</p><ul><li><a href="https://chinadigitaltimes.net/chinese/663402.html">【弄个大新闻】懒人学习刷分工具“Fuck学习强国”宣布停止更新</a></li><li><a href="https://chinadigitaltimes.net/chinese/606936.html">【弄个大新闻】GITHUB上出现“学习强国刷分工具”</a></li></ul><p>近日，fuck-xxqg作者唐文正（<a href="https://2047.name/member/Xmader">@Xmader</a>, <a href="https://github.com/xmader">https://github.com/xmader</a>）选择公开他现实世界的身份。自互联网进入中国以来，中国程序员不断地因为开发抵抗独裁专制的各种软件工具而被当局约谈、警告、罚款、拘留甚至判刑。在这样的政治环境下，唐文正决定公开身份的做法，令许多人感到惊讶和不解。</p><p>以下是本社特派记者thphd对唐文正的文字采访。</p><hr><p><strong>Xmader你好，很高兴认识你。作为一名程序员，如果被学习强国这样的任务困扰，我可能也会开发一个这样的软件。</strong></p><p>我不是全职的程序员，只是一个普通的高中生。</p><p><strong>那你为什么要开发fuck-xxqg？</strong></p><p>我的父母都在体制内工作，2019年时每天有分数要求，这会占用他们每天至少一小时的时间。我希望帮他们从每天重复的这项任务中解脱出来。</p><p><strong>学习强国最令你不能接受的特性是什么？</strong></p><p>公司内部公示排名、部门作为一个整体评比，作为绩效考核的一部分。</p><p><strong>因为什么原因，决定把这个软件公之于众？</strong></p><p>我内心是反习的，而且我想要帮助更多人摆脱对这种意识形态材料的强制性接受、摆脱形式主义。</p><p><strong>为什么你不接受捐赠？</strong></p><p>我不想让人联想到我是以金钱为目的做这个项目的（因为公开到 Github 似乎对我没有直接的益处）；共党不能用经济罪名为幌子处罚我。</p><p><strong>学习强国有各种不同的内容，你认为哪些是值得看的，哪些是洗脑？</strong></p><p>我认为学习强国的所有内容都不值得看。对于客观知识的学习，有更加专业性和更有结构性的 APP 可供选择。这些从来不属于学习强国需要的内容，只是在出现刷分软件后招揽真实用户浏览涉政涉习内容的一种手段。</p><p><strong>你提到你对习近平是持反对态度的，具体是指反对他的哪些做法或者思想？你所说的反习，和一般讲的反共的区别在哪里？</strong></p><p>进行个人集权（“反腐”、修宪、抓律师），思想文化控制（例如文字狱、网络审查、媒体审查、抓公民记者），上行下效拍马屁文化。我认为重要的是要改变这一套党文化。反习反共不是目的只是手段。</p><p><strong>从公布这个软件之后，到今天，心态上发生了什么变化？</strong></p><p>从最开始只是想给身边人用，到后来有了一种社会责任感。现在学习强国加入了视频课等真正的“学习”内容，并且（据说）取消了排名。不得不说，这是一个对付刷分软件非常聪明的手段，但无法改变洗脑的本质。（有了视频课等后，还有人会去浏览习近平相关的内容吗？）</p><p><strong>你说的社会责任感是指什么？</strong></p><p>主动关心和投入社会事件，不一定寻求对自身的益处。</p><p><strong>可不可以认为说你的软件对社会有正面推动作用？</strong></p><p>是的，至少让人重新思考了强迫观看意识形态内容一定时长并半公开排名的合理性，并且可能在一定程度上减缓了社会现状恶化下去。（加速主义者一定不喜欢我这么做）。</p><p><strong>你认为你开发这个软件是合法的吗？是道德的吗？</strong></p><p>当然合法（去除“大不敬罪”）且道德。刷分软件只是模拟浏览器点击和滚动等，不可能有非法侵入计算机系统罪；学习强国和我这个刷分软件都不涉及到金钱，不可能以经济罪行判刑。</p><p><strong>身边其他人对这个软件的看法如何？你认为他们的看法有道理吗？</strong></p><p>我的父母认为我这种行为非常危险（“大不敬罪”），不再使用刷分软件，并且说教让我立刻停止。我理解他们是为自身安全考虑，但是我拒绝共党绑架我的思想和行为。</p><p><strong>大不敬罪是真实存在的吗？大不敬罪在你身边、在你父母身边普遍吗？</strong></p><p>存在，但是一般不会以政治罪名判刑。大不敬罪很普遍，我有一个亲戚因为在微信中提到习近平（具体说了什么我不知道），被叫到公安局要求解释。</p><p><strong>能否举一些大不敬罪给自己招致灾祸的例子？</strong></p><p>许章润、蔡霞、任志强。</p><p><strong>你现在是否担忧你的人身安全？原因是？做了哪些准备？</strong></p><p>当然担心。虽然我已经离开了中国并且决定不再回去，但是我的家人都在中国，我担心共党用他们逼迫我回去。</p><p><strong>你身边的人有没有担忧你的人身安全？</strong></p><p>我认为我的父母没有真正担心我的人身安全，他们甚至想要直接联系中国警方把我从海外“抓”走，换取生活不受影响（真的能吗？）。</p><p><strong>你提到你的父母想要让当局把你带回中国，这是一种猜测，还是你通过某些渠道掌握的？他们有什么动机这么做？我不是很能理解，是对工作和生活有影响吗？</strong></p><p>我和他们仍然有微信视频联络。他们认为我情商低，头脑简单、不成熟，不懂得人情世故；认为做这些事情对自己没有益处，会导致家破人亡。</p><p><strong>你从什么时候开始学习软件设计，动力来自哪里？<br> </strong><br> 从2017年开始。只是个人的兴趣爱好，当时没有想过对自己有什么用处。</p><p><strong>身边人（家人、同学、朋友）是否了解你的软件设计技能，他们怎么看？</strong></p><p>身边人不了解。</p><p><strong>除了fuck-xxqg，你还有哪些活动是涉及政治或者带有政治色彩、表达政治思想的？</strong></p><p>开发免翻墙（hosts，domain fronting，和 ESNI）客户端，<a href="https://github.com/china\-digital\-times" title="中国数字时代内容抓取备份">中国数字时代内容抓取备份</a>。 如果更广义地讲，我在做的音乐相关的项目是为了反对版权控制。唱片公司的利益链条作为中间商层层盘剥，而真正的作者音乐人几乎没有受益（虾米音乐之死是一样的原因）。在图书、论文领域已经有成熟的项目，但是我没有了解到有针对独立音乐人的项目。</p><p><strong>你认为出生在一个体制内的家庭，对于你希望达到的长期目标而言，是一种幸运还是一种遗憾？</strong></p><p>我认为是一种遗憾，共党可以随时以他们的工作作为筹码，要求他们劝说加上逼迫让我回国。</p><p><strong>很多决定不再回中国的人，一般都有一个下决心（思想转变）的过程，相比之下由于你是从体制内出来，会不会需要下更大的决心、作更大的思想转变</strong></p><p>不会啊，我初中时就已经认识到中国社会不公。</p><p><strong>不回国对你意味着要放弃什么？你所放弃的东西，在国外能得到补偿吗？</strong></p><p>不回国意味着我不能面对面见到我的亲人（尤其是老人）。在国外不能得到补偿，但是我认为我的思想自由比亲情牵绊重要得多。</p><p><strong>什么情况下你会考虑回国？</strong></p><p>至少要等到中国民主化之后。</p><p><strong>你崇拜、向往的人或者生活方式？</strong></p><p>Aaron Swartz （当然我不愿意自杀）</p><p><strong>你在Github上的fuck-xxqg项目发布issue的时候不小心用了自己的实名github账号，但你并没有当场删除，而是选择保持公开，为什么？是否后悔过？</strong></p><p>从发布到发现用错帐号中间已经隔了几个小时了。Xmader 这个常用身份和我在现实的关联大，我的名字 Wenzheng Tang 就挂在 Github 主页上，所以直接公开和等别人来找到我或者家人没什么区别，我认为明着公开身份弄个大新闻对我来说更为安全。我不后悔。</p><p><strong>你觉得自己相对于其他同龄人更勇敢吗？</strong></p><p>是的。</p><p><strong>如果你现在不在国外，是否仍然有勇气公开自己身份？</strong></p><p>没有。</p><p><strong>未来这篇采访可能会被许多用过你软件的公务员、事业单位工作人员看到，你希望向他们传达什么？你希望大家记住关于你的哪些事情（希望以怎样的形象存在在大家脑海中）？</strong></p><p>我对 Fuck-XueXiQiangGuo 作者这个身份最初的人设是在体制内科技企业工作的30岁左右的工程师，这可以从早期 issue 的回复中窥探出。 我希望他们不要仅仅因为年龄比他们小，就对我指手画脚。我希望能像 Aaron Swartz 的形象一样。</p><p><strong>未来职业的打算？</strong></p><p>我不想当996社会螺丝钉，我想要当一个 social entrepreneur.</p><p><strong>你的家人也许会看到这篇采访。Anything to say to them?</strong></p><p>如果你们任何人受到连坐，请不要怪罪于我，因为我坚信我做的是正确的事情。就像是被绑匪绑架，正常的思维应当责骂绑匪他太凶恶无情，而不该怨恨我这个惹恼绑匪的人。</p><p><strong>《中国数字时代》前两天有一篇关于你的<a href="https://chinadigitaltimes.net/chinese/663402.html" title="报道">报道</a></strong></p><p>这篇文章大多只是臆测，对我有很多误解。</p><blockquote><p>为什么要停止更新？ 这一项目停更的时间点恰逢两会，所以也有网友担心是否作者遭受了某种外部压力或感知到了危险。开发者停更的原因也可能与某种失望情绪有关。</p></blockquote><p>其实我很早就不更新了，上次更新还是在 2020 年五月。只是想要在项目两周年时正式说一声告别。</p><div class="addtoany_share_save_container addtoany_content
