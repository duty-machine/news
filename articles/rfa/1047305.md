[行业萧条上百名台籍机师自中国返台  台航企还盈利](https://www.rfa.org/mandarin/yataibaodao/gangtai/hx-10282020084334.html)
------

<p> </p><p>台湾镜周刊报导，受新冠肺炎疫情影响，上百名在疫情前到大陆就职的台湾机师, 掀起返回台湾的浪潮。 台湾的华航、长荣航空，因为全球载货需求成长，进入全球航空业营收成长的前四名。 台湾航空公司更锁定国内旅游，发展出“伪出国”产品，突破困境。 <br/> <br/> 台湾航发协会执行长廖玲惠28日接受自由亚洲电台采访，证实在新冠肺炎疫情爆发之前，中国的航空公司高薪挖去台湾机师约有两、三百位，但是自从疫情暴发之后，中国的航空公司开始减少飞行时数、减薪水、减航班，最后进行裁员、资遣。在中国大陆的台籍机师差不多有一半已经陆续返台了。</p><p> </p><ul><li><b><a class="external-link" 中华民国飞行员协会理事长段蓬麟接受自由亚洲电台采访则说，中国大陆大概有一半的航空公司放无薪假，先让机师休个无薪假，先回家修身养性。全世界各国都一样。“我们有在韩国的、香港的、越南、中东、新加坡、日本的都有，大部份都放无薪假，因为航空公司都没有航班可以飞，这是全球航空界所共同面对的问题。”<br/> <br/> 廖玲惠提到，早几年中国大陆提供优厚的条件向台籍机师招手：“比如说以前四川航空会挖角资深机师，差不多飞行时数在3500至4千小时，就会有两倍的薪水，大约是60万新台币。然后满五年送房子、送管家。其他的航空公司有的是直接折成钱，比如北京附近的航空公司的话，有给到一个月80万的都有，70、80万新台币（约16、18万人民币）。”<br/> <br/> 廖玲惠说，中国大陆航空业发展太快，机师的储备量不足，急需大批有足够飞行时数、专业的机长带领副机师，加上台湾机师的英文程度好，比较能胜任国际航班。</p><p> </p><p><div class="image-inline captioned" style="width:640px;"><div style="width:640px;"><img 负责介绍台籍机师到中国大陆航空公司的中华民国飞行员协会理事长段蓬麟则认为，中国籍的航空公司向全球征才，从严考核，不能叫“挖角”，加上当时台湾高铁通车，国内航班萎缩，有几家航空公司破产、裁员，中国大陆薪资水平相较高，成为台籍机师谋职的选项。<br/> <br/> <b>全球航空业遭致命冲击 台湾航空业存活还获利</b><br/> <br/> 廖玲惠说，疫情后，台湾还没传出有机师被资遣、裁员或减薪，空服员底薪3万，没有出国津贴。<br/> <br/> 廖玲惠说，台湾航空业能存活，主要是货机需求量很大，业者并开发出“跳岛”、“伪出国”等新产品，避开14天隔离检疫的问题。<br/> <br/> 廖玲惠解释何谓“伪出国”：“假装出国，台北松山机场绕台湾一圈，松山机场出、松山机场回，是同一个定点，就看台湾周边景色，还能买免税商品，也同样享有飞机餐的伪出国的体验。”</p><p> </p><p><div class="image-inline captioned" style="width:640px;"><div style="width:640px;"><img 林国显提及，今年第二季，全世界仅有4家航空公司获利，台湾的长荣、华航就是获利的公司，加上国旅复甦，观光补贴，使得航空公司能持续运作。<br/> <br/> <b>华航、长荣货机成保命金鸡母  未裁员还继续招募</b><br/> <br/> 华航在全球疫情最严峻时，肩负所谓“天空国家队”的任务，为台湾载运赠送给世界各国1千万片口罩的任务。<br/> <br/> 廖玲惠告诉本台，华航有18架货机，占华航机队四分之一，是全球航空业货机量第六大。长荣航空则有9贺货机，这次台湾在疫情后的货机需求量成长二、三成。</p><p>廖玲惠举例：“送口罩、卖口罩，相关科技远距教学产品，台湾算出口国，生技产品、高科技、远距的高科技产品价值比较高，都用航空货运处理。如果疫苗研发出来，预计需要八千架次的波音七四七载送疫苗到世界各地，货运需求量高。”</p><p> </p><p><div 李信燕认为，之前中国大陆内需市场很大，薪资条件很高，而台湾航空市场人力充足、专业，在台湾退休，符合资格到国外去担任机师，是蛮好的方式，现在大陆内需市场下降，释出一些人力也是理所当然。<br/> <br/> <b>台湾航空业为促销新产品层出不穷</b><br/> <br/>
