[蔡英文与澳洲智库对话 忧心区域擦枪走火](https://www.rfa.org/mandarin/yataibaodao/gangtai/hcm2-08272020113928.html)
------

<p> </p><p>台湾总统蔡英文27日应澳洲智库“澳洲战略政策研究所”(ASPI)邀请，发表视频演说及回答书面提问。当被问及“两岸发生冲突的可能性”时，蔡英文回应，“区域军事活动日益增加，我们担心会擦枪走火。”<br/> <br/> 蔡英文总统是应澳洲智库邀请，在“ASPI印太领袖对话”系列活动发表演说。蔡英文提到，台湾和澳洲是印太地区的重要成员，维护区域的和平与繁荣是台湾的责任，也符合台湾的最佳利益。<br/> <br/> 蔡英文：“我们这个区域情势变化快速，需要自二战以来最全面的战略调整。‘灰色地带活动’就是我们必须共同面对的挑战。”</p><p> </p><ul><li><b><a <br/> 蔡英文在回应书面提问时被问及，“两岸爆发冲突的可能性如何？”蔡英文表示，各方必需审慎约束，才能避免两岸爆发冲突。台湾期盼北京保持自制，克尽身为区域大国的义务。国际社会密切关注香港局势和中国在南海的军事化行动，也因此更严密监控台湾海峡的情势。蔡英文还提到，“区域军事活动日益增加，我们担心会擦枪走火。因此，各方保持开放的沟通管道是很重要的，以避免误解和误判。”<br/> <br/> 与会者也对于台湾如何解读“一中”政策表达关注。蔡英文回应表示，如同习(近平)先生去年初的声明，北京的“一个中国”在本质上就是“一国两制”，这是台湾人民无法接受的，尤其是台湾已经目睹了香港的情况。台湾是个民主蓬勃发展的国家，中华人民共和国对台湾没有管辖权，台湾的公民享有完全的政治权利，台湾拒绝任何矮化台湾的意图。台湾致力维系台海和平稳定，她相信国际社会也都注意到了。</p><p> </p><p><div 也有提问直接了当地问蔡英文，“台湾要寻求独立吗？有可能统一吗？”蔡英文重申，“台湾是台湾，我们的存在无庸置疑。”蔡英文说，台湾有自己的政府，有自由的选举制度，人民可以选择自己的领导人，也有自己的军队和民主制度，不曾接受过北京管辖。<br/> <br/> 蔡英文强调，台湾的未来完全由台湾人民决定，这是我们一直的坚持。她要重申致力台海和平稳定的承诺。只要有助双方互蒙其利，台湾乐意与中国协商，重要的是协商必须是基于“和平、对等、民主、对话”四项原则。如果能秉持这些原则，台湾愿意推动两岸互动。<br/> <br/> 另一方面，美国在台协会在脸书以台美“安全合作月”为名，贴出美国和台湾的安全合作，以及台湾从美国采购战斗机的相关图片。照片包括今年汉光演习中，美方人员共同参与，在爱国者飞弹系统前的合影，以及台湾的F-16飞行员在美国接受空中加油的训练。在台协会表示，美台安全多面向的合作，是对美台双方均有助益的“真朋友，真进展“。 <br/> <br/> <br/> 自由亚洲电台记者黄春梅 台北报道 责编 许书婷
