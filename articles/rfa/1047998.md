[中共对台“首战即终战”？   五成八台湾人不认同](https://www.rfa.org/mandarin/yataibaodao/gangtai/hx1-08242020091817.html)
------

<p> </p><p>台湾前总统马英九上周六在一个研讨会上再次表示，一旦解放军攻台，“首战即终战”，还质疑蔡英文将国家推到战争边缘。蔡英文则反驳马英九的想法和主流民意距离遥远。台湾民意基金会24日公布民调显示，五成八受访台湾人反对马英九“首战即终战”的说法。<br/> <br/> “马英九基金会”上周六（22日）主办“国家不安全研讨会：台湾如何转危为安”，前总统马英九批评蔡英文总统不接受“九二共识”，废弃两岸政治互信与共同政治基础，对外关系向美倾斜，“联美抗陆”使得台湾卷入大国斗争，这些正是让台湾濒临险境的祸因。</p><p> </p><ul><li><b><a class="external-link" ”</p><p>马英九提到，蔡英文曾对英媒BBC记者说，台湾做好军事防卫，让中共侵台“将付出很大代价”；蔡英文还曾向CNN记者表示，期待台湾“承受第一波攻击后”，世界各国能站出来给大陆强烈压力。但马英九警告：“国防安全，不能一厢情愿”。<br/> <br/> 马英九重申他“首战即终战”论点，出自国防部所属的智库国防安全研究院在2018年提出的《中共政军发展评估报告》：“共军攻台的战略是‘首战即决战，让美军未到，战事已定’。换言之，中共打算在第一波攻击后就拿下台湾，形成终战之势。兵凶战危，身为国家领导人，更应避免战祸，虽不畏战但不应求战，这种‘先承受第一击再盼各国来救援’的思维，置台湾人民生命财产安全于何地？”</p><p> </p><p><iframe <b>马英九：要倾尽余生防止两岸战争</b></p><p> </p><p>总统府发言人黄重谚第一时间回应指，北京政府并未接受马英九前总统的“一中各表”说法。马英九认为蔡政府要接受“九二共识”，才能免于中共攻台的“首战即终战”的说法。无论基于个人的乡愁，还是因为对国际局势缺乏认识，身为一位卸任国家元首，马英九前总统不应该和北京一起威胁打压台湾，并指控政府把守护国家主权当作是开战的说法。<br/> <br/> 马英九在22日下午的圆桌论坛上另宣布，从现在起到他告别世界的那一天为止，他会尽力防止两岸战争的发生。马英九强调，“九二共识”就是一个很好的工具，它绝非“马英九的乡愁”，而是“国家的钥匙”。<br/> <br/> <b>苏起：美国驰援台湾鞭长莫及</b><br/> <br/> 出席这场“国家不安全研讨会”的前国安会秘书长苏起则表示，根据澳洲悉尼大学、美国亲保守派智库的公开资料，美国十一艘航空母舰，五艘在东岸、五艘在西岸，一艘维修中，关岛、珍珠港都没有，最近的只有雷根号，最快走到台湾至少十二天。<br/> <br/> 苏起：“两个礼拜，光是航行时间，前面还有决策辩论的时间，还要通知国会，还要准备粮草、准备弹药，两、三个礼拜跑不掉，照这样来看的话，在座很多将军都可以告诉我，台湾早就变天了。”<br/> <br/> 苏起认为，已有犯台能力的中共何时出手无人能预测；他认为美国大选前，中共应该不会躁进，但选后什么事都可能。<br/> <br/> 苏起提到最近美国出现一些声音，说到防守台湾鞭长莫及：“时间那么短，成了既成事实，美国还要赶过来，救台湾根本来不及了。美国就算救了台湾，还要留下来保护台湾，他要花多钱？所以美国人根本不可能来做这件事。台湾老百姓好像很多人没有考虑到。对绿营的人来说，你离你讨厌的大陆太近，你离你喜欢的美国太远，所以美国到时候就是爱莫能助，心有余而力不足。”<br/> <br/> 苏起断言，若台海发生战争，特朗普绝对不会来救台湾。他还说曾有美国军方高层认为“两岸一旦发生战争，台湾一天就会投降”。苏起说，特朗普为美国总统胜选大打“台湾牌”，台湾也全力配合，但中共是否最后会像对付香港一样，加倍奉还？值得关注。<br/> <br/> 苏起还说，台湾为什么表现得比有美国盟约保障的日本及韩国更勇敢，这是因为民进党政策是百分之百反中，而日韩都是既交往又对抗。</p><p> </p><p><div class="image-inline captioned" style="width:1280px;"><div style="width:1280px;"><img alt="4.jpeg" height="820" 对马英九称因蔡英文不愿意接受“九二共识”而把台湾带上战争边缘等言论，蔡英文23日在脸书发文指出，“国际与两岸局势变化迅速，无论是共机绕台、香港反送中、美中贸易战，我们必须比过去更清楚了解，台湾在这个时代所扮演的关键角色。 我想对马前总统说，国际局势瞬息万变，在国家内部寻找敌人并没有意义。我更要请马前总统冷静想想：为什么您的想法，和主流民意有这么远的距离？和国际社会有这么大的落差？身为民主国家领导者，守护主权、民主和自由，让世界看见台湾的决心，这是我决不会退缩的责任。”<br/> <br/> 马英九办公室则回应，无法理解蔡英文总统“在国家内部寻找敌人”之语为何意？在野党对执政党监督、民众对政府施政提建言，都是天经地义。<br/> <br/> <b>外交部：“国防外交无用论”悖离台湾民意</b><br/> <br/> 台湾外交部也批评马英九说，一位曾经统帅三军、领导外交的前总统，和一位曾经主导国家安全的前国安会秘书长，却视国家自我防卫为对敌挑衅，视发展对外关系为对中国威胁，视美国对台友好与支持为无用。这些“国防外交无用论”的论点，已背离台湾社会主流民意。而台湾副总统赖清德也反击表示，前总统马英九不应该还活在过去，更不该一厢情愿。如果不断地造成国内分歧、制造恐慌，这比战争还恐怖。<br/> <br/> 马英九办公室24日再反击说，现今两岸情势高度紧张，被国际危机组织列为“政治安全情势重大恶化地区”，美国与大陆间冲突不断，这些都是事实。蔡总统以国家机器不断制造贩卖芒果干（亡国感），就是靠制造恐慌当选。</p><p><b>台湾最新民调：五成八反对首战即终战</b><br/> <br/> 台湾民意基金会24日发布的一项与“中国军演、台湾政局、美中台关系”相关的民调，其中一题问：中共军方表示将在台湾海峡及南北两端进行实战演习，是九六年飞弹危机后最大规模对台军事行动，担不担心两岸之间爆发战争？有五成八受访者表示不担心，四成一表示会担心。<br/> <br/> 民调另一题，问到马英九日前曾说“首战即终战”，意思是：“两岸一开战，台湾根本等不到美军来，战争就结束，中共就赢了”，是否同意？有五成八表示不同意，三成四同意。<br/> <br/> 对马英九“首战即终战”的说法，与会的前台北市副市长丁庭宇说，“当过总统或现任总统的人，绝对不适合在两岸关系中间，丧失自己的信心。如果国军今天‘首战’即结束的话，那我们还是早点和谈比较好，我不相信。而且我们空军还是有一战的实力。但很明显在我们所看到的状况，现在国军的战力是退缩。”<br/> <br/> 台湾关怀中国人权联盟理事长杨宪宏则呼吁马英九再把话讲清楚一次。“我不相信做过总统的人，说出来的话，会引起近六成的人不同意他。我们从政党战略来看，他讲完这个话以后，不但对民进党一点伤害没有，而且还帮民进党拉抬非常多。那么他伤害了谁？他伤害了江启臣，现在国民党的主席不知道怎么办？如果你们去堵江启臣，问他说，马总统说首战即终战，因为美国不会来驰援，江启臣逃都来不及啊！”</p><p> </p><p><div 急统派的新党青年军，则在24日召开记者会声援马英九。新党“21青”连线发起人苏恒说：“网路上现在已经有人在疯传说所谓的“不参战”声明，还有女网友在网路上询问说，我的男朋友他只当了四个月的兵，他对政治很冷感，他对两岸的走向没有感觉。如果我的男朋友不愿意上战场的话，请问一下，这样有没有犯法的问题？今天你让我们的民众如此担心害怕，需要到到网路上问这种问题的时候，蔡英文你不觉得很丢脸吗？”<br/>
